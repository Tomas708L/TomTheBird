# Overview
This is a Flappy Bird-style game built using Unity. The player controls a bird that must navigate through pipes without hitting obstacles. The game includes a start menu, scoring system, and game over screen.


# Fetures

* Player Controls: Press Space to make the bird flap upward.

* Scoring: Player earns 1 point each time they pass between the pipes.

* Obstacles: Pipes move from right to left, and the player must avoid colliding with them.

* Game Over: Collision with pipes or ground triggers a game over screen.

* Start Menu: Pause before starting, with a Play button to begin the game.

* Restart: Restart button to reload the game after game over.

* Parallax Background: Moving background for visual effect.

  # How to play The Game
* Press Play on the Start Menu to start the game.

* Press Space to make the bird flap upward.

* Avoid hitting The pipes.

* Score points by passing through the middle of the pipes.

* When the bird collides with a pipe or the ground, the game ends.

* Press Restart to play again.
